## Project Description
This repository contains QGIS projects completed as part of Module 2, focused on building foundational desktop GIS skills. I worked through ten tutorials covering core GIS operations including map creation, data management, spatial analysis, and styling. Each tutorial was completed in QGIS and saved as a project file with an accompanying screenshot.

## Tools and Technologies
- QGIS (Desktop GIS)
- GeoPackage / Shapefile formats
- CSV / Spreadsheet data
- Coordinate Reference Systems (CRS)

## What I Did
- Completed 9 required QGIS tutorials covering topics such as projections, attribute tables, vector and raster styling, and spatial joins
- Completed 1 additional tutorial of my choice from the extended tutorial list
- Saved each tutorial as a `.qgz` project file and captured a screenshot of the map canvas with the Layers panel visible
- Organized all project files in a `projects/` folder and screenshots in an `outputs/` folder
- Documented the repository with an overview, software environment, and reflection

## How to View / Run
- Open any `.qgz` file in the `projects/` folder using QGIS (download at [qgis.org](https://qgis.org))
- View screenshots directly in the `outputs/` folder without any software installation
